# Quake SWEPs Enhanced
## A mod my Toad64

### Premise
This mod serves to add a deathmatch gamemode using Upset's QUAKE SWEPs ([url]https://steamcommunity.com/sharedfiles/filedetails/?id=168506010[/url])
The mod also includes a custom HUD, a Steph playermodel ([url]https://steamcommunity.com/sharedfiles/filedetails/?id=3166758238[/url]) and a modifed rocket launcher for rocket jumping :D
