
# Quake SWEPs Enhanced

## A mod by Toad64

### Premise
This mod serves to add a deathmatch gamemode using Upset's [QUAKE SWEPs](https://steamcommunity.com/sharedfiles/filedetails/?id=168506010)
The mod also includes a custom HUD, a [Steph playermodel](https://steamcommunity.com/sharedfiles/filedetails/?id=3166758238) and a modifed rocket launcher for rocket jumping :D
